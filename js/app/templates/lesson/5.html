Lesson #5: align-items

Another parent property, align-items defines how the parent's children will be aligned relative to the cross-axis of the flexbox. align-items can be set to a few attributes we've already seen like flex-start | flex-end | center and a few new ones like stretch | baseline. Mozilla summarizes it below:

The CSS align-items property aligns flex items of the current flex line the same way as justify-content but in the perpendicular direction.
MOZ
tl;dr
  • the opposite of justify-content
  • the default attribute is stretch
instructions

Because we set the height and line-height of the .child elements our style.css file, you'll need to overide the height and line-height properties to see the full effects of align-items: stretch;.

.parent {
  align-items: flex-start | flex-end | center | stretch | baseline;
}

.child {
  height: auto !important;
  line-height: auto !important;
}

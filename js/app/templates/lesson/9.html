Lesson #9: flex-basis

The last sub-property for flex: ..., flex-basis specifies the initial size of the flex-item before anything gets applied to it. Read the following excerpt from this helpful SO answer:

flex-basis allows you to specify the initial/starting size of the element, before anything else is computed. It can either be a percentage or an absolute value.

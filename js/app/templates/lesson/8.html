Lesson #8: flex-shrink

2 out of 3 factors for the main property flex. Most resources I found for this property weren't very useful. It took awhile to figure out how flex-shrink works and when you'd want to use it. Basically if the container is too big for the content, you can tell an element to shrink. Kinda backwards, kinda trippy. Let's play with it.

...specifies the "flex shrink factor", which determines how much the flex item will shrink relative to the rest of the flex items in the flex container when there isn't enough space on the row.
CSS-Tricks
tl;dr
  • when the items are too large for the container
  • default value is 1
instructions

Currently the parent element has a max-width of 400px, but its children elements would all like to be 100px. Because there are 8 children display: flex; sets them all to an equal width: 400/8 = 50, so that's what they are. Change all the children's flex-shrink property to be equal to their nth-child. You'll get a nice cascading effect. Like...

.child:nth-child(1) {
  flex-shrink: 1;
}
.child:nth-child(2) {
  flex-shrink: 2;
}

Lesson #7: flex-grow

We've already kind of used flex-grow before (along with flex-shrink and flex-basis which we'll get to in a second), but I was kind of sneaky about it. flex is actually shorthand for all three. flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]. For now, let's just worry about flex-grow

flex-grow defines the ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion. It dictates what amount of the available space inside the flex container the item should take up.
CSS-Tricks
tl;dr
  • if siblings are set to 1 and el is set to 2 it will take up twice as much space
  • takes a number
  • default is 0
instructions

Set every child element's flex-grow to 1, then set only the first child to 2

.child {
  ...
  flex-grow: 1;
}
.child:first-child {
  flex-grow: 2;
}

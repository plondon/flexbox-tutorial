Lesson #6: order

Finally! Our first child property. order takes an integer, and it is a very powerful and intuitive way to change the order of the child elements. By default, elements will be laid out in the order they are in the DOM, but we can easily change this with the order property. Think of order as grouping the elements, since multiple children can have the same order attribute.

tl;dr
  • child property
  • multiple children can have the same order
